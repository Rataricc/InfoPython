Listas en Python

Las listas en Python son conjuntos de elementos ordenados y pueden estar conformados por
elementos de distintos tipo (enteros, float, cadenas, funciones, otras listas, etc).

Las listas en Python son mutables, lo que significa que sus elementos pueden cambiar o reordenarse.



Las listas se delimitan por corchetes [] y sus elementos se separan por comas.



Los elementos de una lista pueden accederse mediante su índice.

Si queremos conocer la cantidad de elementos en una lista podemos utilizar la función

Podemos acceder a los distintos elementos de la lista por medio de su índice.



Los índices de las listas comienzan en 0 (cero). Por eso el índice 0 para acceder
al primer elemento, el índice 1 para el segundo y así sucesivamente.
También podemos usar de índices enteros negativos. Correspondiendo el -1 al último
elemento, el -2 al anteúltimo y así sucesivamente.



Una lista anidada se considera como un solo elemento, independientemente de cuántos
elementos contenga.

Obtener sublistas (slicing)

Se puede obtener sublistas a partir de una lista usando [:] de esta manera:



El primer número indica el índice del primer elemento, y el segundo en donde
detenerse y dejar de tomar elementos.



Si se decide omitir el primer número, se comenzará desde el inicio.



mi_lista[:4] es igual a hacer mi_lista[0:4]

Si se elimina el segundo número se tomarán elementos hasta el final.



mi_lista[3:] es igual a hacer mi_lista[3:len(lista)]

Ahora si no pones ningún número, y solo deja los dos puntos, se copiará la lista
completa.



mi_lista[:] es igual a mi_lista[0:len(lista)]
