INSTALACIÓN DE PYTHON 3 EN WINDOWS

Primero comprueba si tu ordenador ejecuta la versión 32 bits de Windows o la de 64, en
"Tipo de sistema" en la página de "Acerca de". Para llegar a esta página, intenta uno de
estos métodos:

  • Presione la tecla de Windows y la tecla Pause/Break al mismo tiempo.
  • Abre el Panel de Control desde el menú de Windows, despúes accede a Sistema
    y Seguridad, luego a Sistema.
  • Presiona el botón de Windows, luego accede a Configuración > Sistema > Acerca de.

Puedes descargar Python para Windows desde el siguiente sitio web

Pagina oficial de Python

Paso 1

Hacé clic en el enlace "Latest Python 3 Release - Python x.x.x".

Paso 2

Si tu compu ejecuta la versión de 64 bits de Windows, descarga Windows x86-64 executable installer.
De lo contrario, descarga Windows x86 executable installer. Después de descargar el instalador, deberías ejecutarlo
(dándole doble click) y seguir las instrucciones.

Muy importante para tener en cuenta

Durante la instalación, verás una ventana de "Setup". 
Asegúrate de marcar las casillas "Add Python 3.x to PATH" o 
"Add Python to your environment​ variables"
y hacer clik en "Install Now", como se muestra aquí (
puede que se vea un poco diferente si estás instalando una versión distinta)

Una vez finalizada la instalación, abre una ventana de comandos en Windows (Busca
en Windows la aplicación CMD y ejecútala).

Para conocer sobre variables de entorno haga click en el siguiente enlace.

Entorno de Trabajo

Vale aclarar en este momento, que existen momento, que existen varias maneras de crear programas y ejecutar nuestra código Python, pero en este curso/blog trabajaremos sin utilizar programas extras (como un IDE) para lograr un mejor entendimiento y poder tener mayor control sobre esta tecnología.

Utilizaremos el intérprete (ya instalado en el paso anterior) y la ventana de comandos de Windows para ejecutar nuestro
programa. Y para ecribirlo usaremos cualquier editor de texto que este a nuestra disposición como por ejemplo: Sublime Text,
Visual Studio Code, o tambien podrian a utilizar algun entorno de desarrollo integrado (IDE), como por ejemplo PyCharm...etc.

Enlaces de Descarga: 

Descargar Sublime Text  Descargar Visual Studio Code Descargar PyCharm

Hagamos nuestro primer programa

Para terminar nuestra guía, crearemos nuestro primer programa y lo ejecutaremos, lo único
que realizara es mostrar por pantalla un mensaje de bienvenida.

1. Crearemos un archivo y lo llamamos miPrimerPrograma.py

2. Abriremos el archivo con el editor de texto que tengan, y
escribimos el siguiente código.

3. Guardemos y ya tenemos listo nuestro programa (si asi de sencillo).

4. El paso siguiente es ejecutarlo, para eso abrimos una ventana de comando (CMD de
Windows) y en la misma vamos hasta la ubicación donde hayamos guardado el archivo.

  • Dentro de la ventana siempre veremos la ruta donde actualmente estamos parados, por ejemplo: C:\Users\Fulanito\Desktop , es decir  estamos parados en el escritorio del usuario Fulanito, para moverte entre directorios  debes usar los comandos  cd  nombreDirectorio para ingresar a ese directorio y  cd.. para regresar un nivel. 

Para mas información les invito a que hagan click en el siguiente link: 

Cómo cambiar de directorio en CMD

En el ejemplo se guardó el archivo miPrimerPrograma.py en c:\Users\Usuario\Desktop\Programacion\Programas python>miPrimerPrograma

osea para que entiendan un poco mas, utilice el comando cd para moverme entre los directorios, y los directorios son las carpetas
que tenemos creadas que en este caso Programacion y Programas python son carpetas.

Puede pasar (depende de cómo se realizó la instalación), que se necesario escribir la palabra python
antes de ejecutar el programa, seria algo asi:

c:\Users\Usuario\Desktop\Programacion\Programas python>python miPrimerPrograma
Con el comando python miPrimerPrograma.py, le estamos diciendo que abra el archivo
indicado con el intérprete de python.
